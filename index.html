Covid 19 em Santa Catarina

Esta página foi desenvolvida com a intenção de avaliar o atual estado da pandemia no Estado de Santa Catarina. Em especial para o município de Itajaí.

Os dados apresentados são provenientes da plataforma brasil.io e da API do IBGE. Os dados foram processados em Python, e o código fonte pode ser encontrado no Github
